# <a name="actuals-impact-in-a-time-and-materials-engagement"></a>Impacto de los valores reales en un compromiso de tiempo y materiales

_**Se aplica a:** Project Operations para escenarios basados en recursos/no mantenidos, implementación lite: del acuerdo a la factura proforma_

La siguiente tabla enumera los datos reales de diferentes tipos de transacciones que se crean en varios eventos de un compromiso de tiempo y materiales.

| Evento | Coste real | Dato real de ventas sin facturar | Dato real de ventas facturadas | Ejemplo |
|---|---|---|---|---|
| La hora se ha creado. | No aplicable | No aplicable | No aplicable | <p>Bob Kozack, de la unidad organizativa de Fabrikam US, que tiene una tarifa de coste de 100 dólares estadounidenses (100 USD) por hora, está trabajando en un proyecto denominado "Instalación de brazos en Adatum". Para este proyecto, su tarifa de facturación contratada es de 200 USD por hora. Aquí hay una entrada de tiempo de muestra de Bob Kozak:</p><p>Bob Kozack, 8 horas</p> |
| El tiempo se envía. | No aplicable | No aplicable | No aplicable | Se crean una línea de diario de costes y diarios de ventas no facturadas para la entrada de tiempo. La tasa de coste y el precio predeterminado se introducen en el movimiento de diario. |
| La entrada de tiempo se recupera antes de que se apruebe. | No aplicable | No aplicable | No aplicable | |
| El tiempo está aprobado: las horas enviadas equivalen a horas facturables. | Se crea un dato real de coste. | Se crea un dato real de ventas no facturadas. | No aplicable | <p>Nuevos datos reales que se crean:</p><ul><li>**Coste real:** Bob Kozack, 8 horas, 800 USD</li><li>**Ventas no facturadas reales:** Bob Kozack, 8 horas, 1600 USD</li></ul> |
| El tiempo está aprobado: las horas facturables se reducen por debajo de las horas enviadas. | Se crea un dato real de coste. | <p>Se crean dos datos reales de ventas no facturadas:</p><ul><li>Dato real de ventas sin facturar cobrables para las horas facturables</li><li>Dato real de ventas sin facturar no cobrables para la cantidad restante</li></ul> | No aplicable | <p>Nuevos datos reales que se crean:</p><ul><li>**Coste real:** Bob Kozack, 8 horas, 800 USD</li><li>**Dato real de ventas no facturadas:** Bob Kozack, 6 horas, 1200 USD, *Cobrable*</li><li>**Dato real de ventas no facturadas:** Bob Kozack, 2 horas, 400 USD, *No cobrable*</li></ul> |
| El tiempo está aprobado: las horas facturables se incrementan por encima de las horas enviadas. | Se crea un dato real de coste. | Se crea un dato real de ventas no facturadas. | No aplicable | <p>Nuevos datos reales que se crean:</p><ul><li>**Coste real:** Bob Kozack, 8 horas, 800 USD</li><li>**Ventas no facturadas reales:** Bob Kozack, 10 horas, 2000 USD</li></ul> |
| Se cancela la aprobación de tiempo. | <p>El estado de ajuste del dato real de coste original se actualiza a **Ajustado**.</p><p>Se crea un coste real de reversión que tiene un estado de ajuste de **No ajustable**.</p> | <p>El estado de ajuste del dato real de ventas no facturadas se actualiza a **Ajustado**.</p><p>Se crea un dato real de reversión de ventas no facturadas que tiene un estado de ajuste de **No ajustable**.</p> | No aplicable | <p>Datos reales existentes que se actualizan:</p><ul><li>**Coste real:** Bob Kozack, 8 horas, 800 USD, *Ajustado*</li><li>**Dato real de ventas no facturadas:** Bob Kozack, 8 horas, 1600 USD, *Ajustado*</li></ul><p>Nuevos datos reales que se crean para revertir el impacto financiero anterior:</p><ul><li>**Coste real:** Bob Kozack, (8 horas), (800 USD), *No ajustable*</li><li>**Dato real de ventas no facturadas:** Bob Kozack, (8 horas), (1600 USD), *No ajustable*</li></ul> |
| La entrada de tiempo se recupera después de que se apruebe. | <p>El estado de ajuste del dato real de coste original se actualiza a **Ajustado**.</p><p>Se crea un coste real de reversión que tiene un estado de ajuste de **No ajustable**.</p> | <p>El estado de ajuste del dato real de ventas no facturadas se actualiza a **Ajustado**.</p><p>Se crea un dato real de reversión de ventas no facturadas que tiene un estado de ajuste de **No ajustable**.</p> | No aplicable | <p>Datos reales existentes que se actualizan:</p><ul><li>**Coste real:** Bob Kozack, 8 horas, 800 USD, *Ajustado*</li><li>**Dato real de ventas no facturadas:** Bob Kozack, 8 horas, 1600 USD, *Ajustado*</li></ul><p>Nuevos datos reales que se crean para revertir el impacto financiero anterior:</p><ul><li>**Coste real:** Bob Kozack, (8 horas), (800 USD), *No ajustable*</li><li>**Dato real de ventas no facturadas:** Bob Kozack, (8 horas), (1600 USD), *No ajustable*</li></ul> |
| El contrato está confirmado. | <p>El estado de ajuste de los datos reales de coste original se actualiza a **Ajustado**.</p><p>Se crean datos reales de coste de reversión que tiene un estado de ajuste de **No ajustable**.</p><p>Los nuevos costes reales se crean después de reevaluar las reglas contractuales.</p> | <p>El estado de ajuste de los anteriores datos reales de ventas no facturadas se actualiza a **Ajustado**.</p><p>Se crean un dato reales de reversión de ventas no facturadas que tienen un estado de ajuste de **No ajustable**.</p><p>Los nuevos costes reales de ventas no facturadas se crean después de reevaluar las reglas contractuales.</p> | No aplicable | <p>Datos reales existentes que se actualizan:</p><ul><li>**Coste real:** Bob Kozack, 8 horas, 800 USD, *Ajustado*</li><li>**Dato real de ventas no facturadas:** Bob Kozack, 8 horas, 1600 USD, *Ajustado*</li></ul><p>Nuevos datos reales que se crean para revertir el impacto financiero anterior:</p><ul><li>**Coste real:** Bob Kozack, (8 horas), (800 USD), *No ajustable*</li><li>**Dato real de ventas no facturadas:** Bob Kozack, (8 horas), (1600 USD), *No ajustable*</li></ul><p>Nuevos datos reales que se crean para el impacto financiero reevaluado:</p><ul><li>**Coste real:** Bob Kozack, 8 horas, 800 USD</li><li>**Ventas no facturadas reales:** Bob Kozack, 8 horas, 1600 USD</li></ul> |
| Se crea una factura. | No aplicable | No aplicable | No aplicable | |
| Se confirma la factura. No hay cambios en la cantidad del detalle de la línea de factura con respecto a la cantidad del dato real de ventas no facturadas. | No aplicable | <p>Se actualiza el estado de la factura de las anteriores ventas no facturadas.</p><p>Se crean un dato reales de reversión de ventas no facturadas que tienen un estado de ajuste de **No ajustable**. | Se crea un dato real de ventas facturadas. | <p>Dato real existente que permanece sin cambios:</p><ul><li>**Coste real:** Bob Kozack, 8 horas, 800 USD</li></ul><p>Dato real existente que se actualiza:</p><ul><li>**Dato real de ventas no facturadas:** Bob Kozack, 8 horas, 1600 USD, *Factura de cliente registrada*</li></ul>Nuevo dato real que se crea para revertir el trabajo financiero en curso (WIP):</p><ul><li>**Dato real de ventas no facturadas:** Bob Kozack, (8 horas), (1600 USD)</li></ul><p>Nuevo dato real que se crea para registrar los valores de ventas facturados:</p><ul><li>**Dato real de ventas facturadas:** Bob Kozack, 8 horas, 1600 USD</li></ul> |
| La factura se confirma después de que la cantidad del detalle de la línea de factura se disminuye con respecto a la cantidad del dato real de ventas no facturadas. | No aplicable | <p>El estado de ajuste de los datos reales de ventas no facturadas se actualiza a **Ajustado**.</p><p>Los datos reales de reversión de ventas no facturadas se crean para los datos reales de ventas no facturadas originales. Tienen un estado de ajuste de **No ajustable**.</p><p>Se crean dos datos reales de nuevas ventas no facturadas:</p><ul><li>Dato real de ventas sin facturar cobrables para las horas facturables</li><li>Dato real de ventas sin facturar no cobrables para la cantidad restante</li></ul><p>Los datos reales de reversión de ventas no facturadas se crean para dos nuevos datos reales de ventas no facturadas.</p> | <p>Se crean dos datos reales de ventas facturadas:</p><ul><li>Dato real de ventas facturadas para las horas facturables</li><li>Dato real de ventas facturadas no cobrables para la cantidad restante</li></ul> | <p>Dato real existente que permanece sin cambios:</p><ul><li>**Coste real:** Bob Kozack, 8 horas, 800 USD</li></ul><p>Dato real existente que se actualiza:</p><ul><li>**Dato real de ventas no facturadas:** Bob Kozack, 8 horas, 1600 USD, *Ajustado*</li></ul><p>Nuevo dato real que se crea para revertir el WIP financiero anterior:</p><ul><li>**Dato real de ventas no facturadas:** Bob Kozack, (8 horas), (1600 USD), *No ajustable*</li></ul><p>Nuevos datos reales que se crean para registrar el WIP de ventas actualizado:</p><ul><li>**Dato real de ventas no facturadas:** Bob Kozack, 6 horas, 1200 USD, *Cobrable*</li><li>**Dato real de ventas no facturadas:** Bob Kozack, 2 horas, 400 USD, *No cobrable*</li></ul><p>Nuevos datos reales que se crean para revertir el WIP de ventas actualizado:</p><ul><li>**Dato real de ventas no facturadas:** Bob Kozack, (6 horas), (1200 USD), *Cobrable*</li><li>**Dato real de ventas no facturadas:** Bob Kozack, (2 horas), (400 USD), *No cobrable*</li></ul><p>Nuevos datos reales que se crean para registrar los valores de ventas facturados:</p><ul><li>**Dato real de ventas facturadas:** Bob Kozack, 6 horas, 1200 USD, *Cobrable*</li><li>**Dato real de ventas facturadas:** Bob Kozack, 2 horas, 400 USD, *No cobrable*</li></ul> |
| La factura se confirma después de que la cantidad del detalle de la línea de factura se incrementa con respecto a la cantidad del dato real de ventas no facturadas. | No aplicable | <p>El estado de ajuste de los datos reales de ventas no facturadas se actualiza a **Ajustado**.</p><p>Los datos reales de reversión de ventas no facturadas se crean para los datos reales de ventas no facturadas originales. Tienen un estado de ajuste de **No ajustable**.</p><p>Se crean nuevos datos reales de ventas sin facturar para la nueva cantidad.</p><p>Los datos reales de reversión de ventas no facturadas se crean para los nuevos datos reales de ventas no facturadas.</p> | Se crean nuevos datos reales de ventas facturadas para la nueva cantidad. | <p>Dato real existente que permanece sin cambios:</p><ul><li>**Coste real:** Bob Kozack, 8 horas, 800 USD</li></ul><p>Dato real existente que se actualiza:</p><ul><li>**Dato real de ventas no facturadas:** Bob Kozack, 8 horas, 1600 USD, *Ajustado*</li></ul><p>Nuevo dato real que se crea para revertir el WIP financiero anterior:</p><ul><li>**Dato real de ventas no facturadas:** Bob Kozack, (8 horas), (1600 USD), *No ajustable*</li></ul><p>Nuevo dato real que se crea para registrar el WIP de ventas actualizado:</p><ul><li>**Dato real de ventas no facturadas:** Bob Kozack, 10 horas, 2000 USD, *Cobrable*</li></ul><p>Nuevo dato real que se crea para revertir el WIP de ventas actualizado:</p><ul><li>**Dato real de ventas no facturadas:** Bob Kozack, (10 horas), (2000 USD), *Cobrable*, *No ajustable*</li></ul><p>Nuevo dato real que se crea para registrar los valores de ventas facturados:</p><ul><li>**Dato real de ventas facturadas:** Bob Kozack, 10 horas, 2000 USD, *Cobrable*</li></ul> |
| Se corrige la factura para disminuir la cantidad o el precio imputable. | No aplicable | <p>Se crean dos datos reales de ventas no facturadas:</p><ul><li>Ventas reales no facturadas cobrables por la cantidad en la factura correctiva</li><li>Dato real de ventas facturables para la cantidad restante</li></ul><p>Los datos reales de reversión de ventas no facturadas se crean para dos nuevos datos reales de ventas no facturadas.</p> | <p>Se crean datos reales de reversión de ventas facturadas.</p><p>Se crean nuevos datos reales de ventas facturadas para la nueva cantidad. | <p>Datos reales existentes que permanecen sin cambios:</p><ul><li>**Coste real:** Bob Kozack, 8 horas, 800 USD</li><li>**Dato real de ventas no facturadas:** Bob Kozack, 8 horas, 1600 USD, *Factura de cliente registrada*</li><li>**Dato real de ventas no facturadas:** Bob Kozack, (8 horas), (1600 USD)</li></ul><p>Dato real existente que se actualiza:</p><ul><li>**Dato real de ventas facturadas:** Bob Kozack, (8 horas), (1600 USD), *Ajustado*</li></ul><p>Nuevo dato real que se crea para revertir los anteriores valores de ventas facturados:</p><ul><li>**Dato real de ventas facturadas:** Bob Kozack, (8 horas), (1600 USD), *No ajustable*</li></ul><p>Nuevos datos reales que se crean para registrar el WIP de ventas corregido:</p><ul><li>**Dato real de ventas no facturadas:** Bob Kozack, 6 horas, 1200 USD, *Cobrable*, *Factura de cliente registrada*</li><li>**Dato real de ventas no facturadas:** Bob Kozack, 2 horas, 400 USD, *Cobrable*</li></ul><p>Nuevo dato real que se crea para revertir el WIP de ventas corregido:</p><ul><li>**Dato real de ventas no facturadas:** Bob Kozack, (6 horas), (1200 USD), *Cobrable*, *No ajustable*</li></ul><p>Nuevo dato real que se crea para registrar los valores de ventas corregidos:</p><ul><li>**Dato real de ventas facturadas:** Bob Kozack, 6 horas, 1200 USD, *Cobrable*</li></ul> |
| Se corrige la factura para incrementar la cantidad o el precio imputable. | No aplicable | <p>Se crean nuevos datos reales de ventas sin facturar para la nueva cantidad.</p> <p>Los datos reales de reversión de ventas no facturadas se crean para los nuevos datos reales de ventas no facturadas.</p> | <p>Se crean datos reales de reversión de ventas facturadas.</p>Se crean nuevos datos reales de ventas facturadas para la nueva cantidad.</p> | <p>Datos reales existentes que permanecen sin cambios:</p><ul><li>**Coste real:** Bob Kozack, 8 horas, 800 USD</li><li>**Dato real de ventas no facturadas:** Bob Kozack, 8 horas, 1600 USD, *Factura de cliente registrada*</li><li>**Dato real de ventas no facturadas:** Bob Kozack, (8 horas), (1600 USD)</li></ul><p>Dato real existente que se actualiza:</p><ul><li>**Dato real de ventas facturadas:** Bob Kozack, (8 horas), (1600 USD), *Ajustado*</li></ul><p>Nuevo dato real que se crea para revertir los anteriores valores de ventas facturados:</p><ul><li>**Dato real de ventas facturadas:** Bob Kozack, (8 horas), (1600 USD), *No ajustable*</li></ul><p>Nuevo dato real que se crea para registrar el WIP de ventas corregido:</p><ul><li>**Dato real de ventas no facturadas:** Bob Kozack, 10 horas, 2000 USD, *Cobrable*, *Factura de cliente registrada*</li></ul><p>Nuevo dato real que se crea para revertir el WIP de ventas corregido:</p><ul><li>**Dato real de ventas no facturadas:** Bob Kozack, (10 horas), (2000 USD), *Cobrable*</li></ul><p>Nuevo dato real que se crea para registrar los valores de ventas corregidos:</p><ul><li>**Dato real de ventas facturadas:** Bob Kozack, 10 horas, 2000 USD, *Cobrable*</li></ul> |
