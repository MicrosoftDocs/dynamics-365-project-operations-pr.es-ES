# <a name="set-up-and-use-pay-when-paid-vendor-payments"></a><span data-ttu-id="9b5de-103">Configurar y utilizar pagos de proveedores de pago cuando se pagan</span><span class="sxs-lookup"><span data-stu-id="9b5de-103">Set up and use pay-when-paid vendor payments</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="9b5de-104">Cuando aprueba a un proveedor para trabajar como subcontratista, es posible que desee retener el pago al proveedor hasta que su cliente le pague por el proyecto.</span><span class="sxs-lookup"><span data-stu-id="9b5de-104">When you approve a vendor to work as a subcontractor, you might want to withhold payment to the vendor until your customer pays you for the project.</span></span> <span data-ttu-id="9b5de-105">Para respaldar este escenario, puede configurar términos de pago cuando se paga (PWP) cuando configura la orden de compra (PO) con el proveedor.</span><span class="sxs-lookup"><span data-stu-id="9b5de-105">To support this scenario, you can set up pay-when-paid (PWP) terms when you set up the purchase order (PO) with the vendor.</span></span>

<span data-ttu-id="9b5de-106">Por ejemplo, cuando un cliente paga un monto en una factura de proyecto, puede liberar parte o todo el monto de la factura del proveedor.</span><span class="sxs-lookup"><span data-stu-id="9b5de-106">For example, when a customer pays an amount on a project invoice, you can release some or all of the vendor invoice amount.</span></span> <span data-ttu-id="9b5de-107">Simplemente configure los términos de PWP que especifiquen que se le pagará al proveedor después de que usted reciba un porcentaje del pago relacionado del cliente.</span><span class="sxs-lookup"><span data-stu-id="9b5de-107">Just set up PWP terms that specify that the vendor will be paid after you receive a percentage of the related payment from the customer.</span></span> <span data-ttu-id="9b5de-108">Si recibe un pago parcial de un cliente, puede liberar manualmente algunas de las facturas de proveedor relacionadas para el pago.</span><span class="sxs-lookup"><span data-stu-id="9b5de-108">If you receive partial payment from a customer, you can manually release some of the related vendor invoices for payment.</span></span>

<span data-ttu-id="9b5de-109">El siguiente ejemplo describe el proceso cuando se utilizan términos de PWP.</span><span class="sxs-lookup"><span data-stu-id="9b5de-109">The following example outlines the process when PWP terms are used.</span></span>

## <a name="example"></a><span data-ttu-id="9b5de-110">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="9b5de-110">Example</span></span>

<span data-ttu-id="9b5de-111">Su organización acepta proporcionar a un cliente 100 ordenadores que tengan software instalado, por un precio de 150,00 dólares estadounidenses (USD) por ordenador.</span><span class="sxs-lookup"><span data-stu-id="9b5de-111">Your organization agrees to provide a customer with 100 computers that have software installed, for a price of 150.00 US dollars (USD) per computer.</span></span> <span data-ttu-id="9b5de-112">Luego, contrata a un proveedor para que le proporcione los ordenadores que tienen el software instalado.</span><span class="sxs-lookup"><span data-stu-id="9b5de-112">You then hire a vendor to provide you with the computers that have software installed.</span></span> <span data-ttu-id="9b5de-113">Según el acuerdo, el cliente debe aprobar la calidad de las computadoras antes de pagar a su organización.</span><span class="sxs-lookup"><span data-stu-id="9b5de-113">According to the agreement, the customer must approve the quality of the computers before it pays your organization.</span></span> <span data-ttu-id="9b5de-114">La política de su organización es retener el pago a los proveedores hasta que el cliente le haya pagado.</span><span class="sxs-lookup"><span data-stu-id="9b5de-114">Your organization's policy is to withhold payment to vendors until the customer has paid you.</span></span> <span data-ttu-id="9b5de-115">Por lo tanto, configura el proyecto para que tenga un porcentaje de PWP del 100 por ciento.</span><span class="sxs-lookup"><span data-stu-id="9b5de-115">Therefore, you set up the project so that it has a PWP percentage of 100 percent.</span></span>

<span data-ttu-id="9b5de-116">El proveedor le envía los 100 ordenadores que tienen software instalado, junto con una factura por 10 000,00 USD.</span><span class="sxs-lookup"><span data-stu-id="9b5de-116">The vendor sends you the 100 computers that have software installed, together with an invoice for 10,000.00 USD.</span></span> <span data-ttu-id="9b5de-117">Debido a que los términos de PWP están configurados para su proyecto, no le paga al proveedor al recibir las computadoras.</span><span class="sxs-lookup"><span data-stu-id="9b5de-117">Because PWP terms are set up for your project, you don't pay the vendor upon receipt of the computers.</span></span> <span data-ttu-id="9b5de-118">En su lugar, envía los ordenadores al cliente, junto con una factura por 15 000,00.</span><span class="sxs-lookup"><span data-stu-id="9b5de-118">Instead, you send the computers to the customer, together with an invoice for 15,000.00.</span></span> <span data-ttu-id="9b5de-119">El cliente inspecciona los ordenadores y aprueba el monto total de la factura del proyecto.</span><span class="sxs-lookup"><span data-stu-id="9b5de-119">The customer inspects the computers and approves the full amount of the project invoice.</span></span>

<span data-ttu-id="9b5de-120">Después de recibir el pago completo del cliente, le paga al proveedor 10 000,00, el monto total de la factura del proveedor.</span><span class="sxs-lookup"><span data-stu-id="9b5de-120">After you receive the full payment from the customer, you pay the vendor 10,000.00, the full amount of the vendor invoice.</span></span>

<span data-ttu-id="9b5de-122">Cuando configura los términos de PWP para un proyecto, debe especificar, como un porcentaje, la cantidad mínima que un cliente debe pagarle por el proyecto antes de pagarle al proveedor.</span><span class="sxs-lookup"><span data-stu-id="9b5de-122">When you set up PWP terms for a project, you must specify, as a percentage, the minimum amount that a customer must pay you for the project before you will pay the vendor.</span></span> <span data-ttu-id="9b5de-123">La cantidad de umbral se calcula automáticamente en las facturas de los clientes para el proyecto.</span><span class="sxs-lookup"><span data-stu-id="9b5de-123">The threshold amount is automatically calculated on the customer invoices for the project.</span></span> <span data-ttu-id="9b5de-124">Siga estos pasos para configurar el porcentaje de umbral para los términos de PWP.</span><span class="sxs-lookup"><span data-stu-id="9b5de-124">Follow these steps to set up the threshold percentage for PWP terms.</span></span>

3. <span data-ttu-id="9b5de-128">En el campo **Código de cuenta**, seleccione una de las siguientes opciones:</span><span class="sxs-lookup"><span data-stu-id="9b5de-128">In the **Account code** field, select one of the following options:</span></span>

    - <span data-ttu-id="9b5de-129">**Tabla**: los términos de PWP se aplican a un solo proveedor.</span><span class="sxs-lookup"><span data-stu-id="9b5de-129">**Table** – The PWP terms apply to a single vendor.</span></span>
    - <span data-ttu-id="9b5de-130">**Grupo**: los términos de PWP se aplican a todos los proveedores de un grupo de proveedores.</span><span class="sxs-lookup"><span data-stu-id="9b5de-130">**Group** – The PWP terms apply to all vendors in a vendor group.</span></span>
