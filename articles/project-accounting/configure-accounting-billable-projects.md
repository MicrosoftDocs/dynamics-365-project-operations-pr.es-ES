# <a name="configure-accounting-for-billable-projects"></a><span data-ttu-id="59f2a-103">Configurar la contabilidad para proyectos facturables</span><span class="sxs-lookup"><span data-stu-id="59f2a-103">Configure accounting for billable projects</span></span>

- <span data-ttu-id="59f2a-106">**Transacciones de tiempo y material**: estas transacciones se facturan a medida que avanza el trabajo en función del consumo de horas, gastos, elementos o tarifas del proyecto.</span><span class="sxs-lookup"><span data-stu-id="59f2a-106">**Time and material transactions**: These transactions are invoiced as the work progresses based on the consumption of hours, expenses, items, or fees on the project.</span></span> <span data-ttu-id="59f2a-107">Estos costes de la transacción se pueden combinar con los ingresos de cada transacción, y el proyecto se factura a medida que avanza el trabajo.</span><span class="sxs-lookup"><span data-stu-id="59f2a-107">These transaction costs can be matched with the revenue on each transaction and the project is invoiced as work progresses.</span></span> <span data-ttu-id="59f2a-108">Los ingresos del proyecto también se pueden acumular en el momento en que se produce la transacción.</span><span class="sxs-lookup"><span data-stu-id="59f2a-108">Project revenue can be also accrued at the time when the transaction occurs.</span></span> <span data-ttu-id="59f2a-109">Durante la facturación, se reconocen los ingresos y, si corresponde, se revierten los ingresos acumulados.</span><span class="sxs-lookup"><span data-stu-id="59f2a-109">During invoicing, revenue is recognized and if applicable, accrued revenue is reversed.</span></span>
